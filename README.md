
# Pinzon-Blog
Pinzon-Blog is a free blog template made with nodejs express and mongodb

## Installation

Use the git bash to install Pinzon-Blog.

```bash
git clone https://github.com/BugraUmut/pinzon-blog
```

## Prerequisites

- Nodejs
- Express
- MongoDB

##Â Usage

Setup .env

MONGODB_URL=your_mongodb_connection_url
